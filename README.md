# AY-3-8912 Emulator (AVR-AY)
The replacement board for the famous AY-3-8910/AY-3-8912 sound chip of the ZX-Spectrum 128 and others. Based on the schematic and firmware on https://www.avray.ru, https://zx-pk.ru/threads/10510-emulyator-ay-8910-na-atmega.html and https://github.com/andykarpov/turbosound28p resources.
